## Overview

Awesome Copilot is a community-curated marketplace of 70 plugins, agents, hooks, and skills for GitHub Copilot. Plugins install without additional setup and are accessible from both the Copilot CLI and VS Code. The directory spans development toolkits, cloud infrastructure, enterprise platforms, testing, observability, and productivity tools.

## Access and Installation

Plugins are discoverable and installable through two surfaces:

- **GitHub Copilot CLI**: browse with `/plugin marketplace browse awesome-copilot`; install with `copilot plugin install <plugin-name>@awesome-copilot`
- **VS Code**: search `@agentPlugins` in Extensions, or use Command Palette → `Chat: Plugins`

No separate account or configuration is required beyond an active GitHub Copilot subscription.

## Plugin Categories

### Development & MCP Servers

Language-specific Model Context Protocol (MCP) development toolkits covering C#, Go, Java, Kotlin, PHP, Python, Ruby, Rust, Swift, and TypeScript. MCP plugins surface language-appropriate context to the Copilot agent during coding tasks.

### Cloud & Infrastructure

Azure cloud development, Oracle-to-PostgreSQL migration assistance, general infrastructure management, and deployment automation.

### Web & Frontend Development

React 18/19 upgrade toolkits, multi-framework support (React, Angular, Vue), and TypeScript-focused assistance.

### Enterprise Platforms

Microsoft ecosystem plugins: Salesforce development, Power Apps, Power BI, Power Automate, Microsoft 365 Copilot integration, and Dataverse solutions.

### Testing & Quality

Automated test generation, unit test scaffolding, TDD workflow support, and security/accessibility compliance checks.

### Observability & Analysis

LLM observability integrations (Arize, Phoenix), software composition analysis (CAST Imaging), and hallucination detection (DoubleCheck).

### Productivity & Collaboration

Napkin for visual whiteboarding, Roundup for status briefing generation, and accessibility-oriented plugins targeting non-technical users.

## Contribution

Community members submit new plugins through the awesome-copilot GitHub repository. The directory is open to agents, hooks, and skills — not only traditional extensions.
