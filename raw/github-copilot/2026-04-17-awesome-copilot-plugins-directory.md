# Awesome GitHub Copilot — Plugins Directory

> Source: https://awesome-copilot.github.com/plugins/
> Collected: 2026-04-17
> Published: Unknown

## Overview

A curated marketplace of 70 plugins, agents, hooks, and skills designed for specific GitHub Copilot workflows. The collection is community-driven and accessible directly through GitHub Copilot CLI and VS Code without additional setup.

## Access Methods

- **GitHub Copilot CLI**: `/plugin marketplace browse awesome-copilot`
- **VS Code**: `@agentPlugins` in Extensions search or Command Palette → `Chat: Plugins`
- **Installation**: `copilot plugin install <plugin-name>@awesome-copilot`

## Plugin Categories

### Development & MCP Servers

Multiple language-specific Model Context Protocol (MCP) development toolkits including C#, Go, Java, Kotlin, PHP, Python, Ruby, Rust, Swift, and TypeScript.

### Cloud & Infrastructure

Azure cloud development, Oracle-to-PostgreSQL migration, infrastructure management, and deployment tools.

### Web & Frontend Development

React 18/19 upgrade toolkits, modern frontend frameworks (React, Angular, Vue), and TypeScript support.

### Enterprise Platforms

Salesforce development, Power Apps, Power BI, Power Automate, Microsoft 365 Copilot integration, and Dataverse solutions.

### Testing & Quality

Testing automation, unit test generation, TDD support, and security/accessibility best practices.

### Observability & Analysis

LLM observability tools (Arize, Phoenix), software analysis (CAST Imaging), and hallucination detection (DoubleCheck).

### Productivity & Collaboration

Visual whiteboarding (Napkin), status briefing generation (Roundup), and accessibility features for non-technical users.
